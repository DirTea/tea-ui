# 计数

## 基础用法
:::demo
```vue
<template>
  <tea-count-to :end-val="894158"></tea-count-to>
</template>
```
:::

## 属性
| 参数     | 说明     | 类型   | 可选值 | 是否必填 | 默认值 |
| -------- | -------- | ------ | ------ | -------- | ------ |
| startVal | 开始值   | number | -      | -        | 0      |
| endVal   | 结束值   | number | -      | 是       | -      |
| duration | 动画时长 | number | -      | -        | 1000   |
