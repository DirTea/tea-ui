# 按钮

## 基础用法

:::demo

```vue
<template>
  <tea-button>默认按钮</tea-button>
</template>
```

:::

## 属性

| 参数       | 说明        | 类型      | 可选值 | 是否必填 | 默认值   |
| -------- | --------- | ------- | --- | ---- | ----- |
| disabled | 按钮是否为禁用状态 | boolean | -   | -    | false |

## 插槽

| 插槽名     | 说明      |
| ------- | ------- |
| default | 自定义默认内容 |
